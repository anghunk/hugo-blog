### 1. Web Archive - 基于 Cloudflare 的网页归档工具

https://github.com/Ray-D-Song/web-archive

Web Archive 是基于 Cloudflare 的免费网页归档和共享服务，将网页保存为单个 html 文件并上传到服务器，服务器接收浏览器插件上传的 html 文件，并存储到数据库和存储桶中，最终在 Web 客户端显示这些文件。
