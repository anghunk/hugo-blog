记录 vite 中引入 tailwindcss

04 Sep, 2024

安装依赖并生成 tailwind.config.js 和 postcss.config.js。

1yarn add -D tailwindcss postcss autoprefixer
2npx tailwindcss init -p

注意，配置中要使用 ES6 模块语法 export default。

 1// tailwind.config.js
 2export default {
 3  content: [
 4    "./index.html",
 5    "./src/**/*.{vue,js,ts,jsx,tsx}",
 6  ],
 7  theme: {
 8    extend: {},
 9  },
10  plugins: [],
11};

添加 Tailwind 的 CSS 文件。

1/* src/index.css */
2@tailwind base;
3@tailwind components;
4@tailwind utilities;

在 main.js 中引入。

1// main.js
2import './index.css';
