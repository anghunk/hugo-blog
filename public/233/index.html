serve00 如何登录？

14 Aug, 2024

很多人在申请完 serve00 后无法登录，简单的介绍一下，首先会收到一封邮件，上面有：

1Login: xxx
2Password: xxx
3SSH/SFTP server address: xxx

打开一个终端工具，比如 xshell，

然后点击【用户身份验证】，选择 password，将账号密码输入，

点击连接后，会出现一个 SSH 身份验证，Keyboard Interactive(l) ，然后把 password 填入即可。
