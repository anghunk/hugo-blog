使用 js 随机生成背景颜色

29 Sep, 2021

先写一个简单的 html 页面出来

1<style>
2  /* ... */
3</style>
4
5<ul>
6  <li>html</li>
7  <li>css</li>
8  <li>js</li>
9</ul>

通过 Math.random() 属性可以随机生成一个数字，然后通过转化为十六进制的方法进行处理，下面就是随机生成 6 位数字的代码，并进行转化的代码。

1const randomHex = () => `#${Math.floor(Math.random() * 0xffffff).toString(16).padEnd(6, "0")}`;

可以在控制台输出看一下结果，随机生成了一个结果 #62113b。

接下来就需要将这个颜色传给上面的 li 标签，首先需要将所有的 li 遍历然后生成一个数组，可以使用 for 循环进行遍历。

1var arr = document.getElementsByTagName('li'), temp = [];
2for (var i = 0; i < arr.length; i++) {
3  temp.push(arr[i].innerHTML);
4}

然后再对数组进行循环处理，每个 li 标签要要运行一遍随机生成颜色，并且传给自身。同样也是通过 for 循环进行操作。

1var li = document.getElementsByTagName("li");
2for (var i = 0; i < li.length; i++) {
3  
4  for (var num = 0; num < li.length; num++) {
5    li[num].style.background = randomHex();
6  }
7
8}

这些事件都是在页面加载完成之后运行的，所以需要通过 window.onload 写入事件，然后把所有的代码合在一起。
