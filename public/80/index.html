通过 getTimezoneOffset() 方法判断当前时区是否为夏令时

22 Oct, 2021

getTimezoneOffset() 方法 方法返回 UTC 时间和本地时间之间的时差，以分钟为单位。

世界协调时间 (UTC) 是世界时间标准设定的时间，UTC 时间与 GMT 时间（格林威治时间）相同。

测一下我所在时区和标准时区的时间差

1function myDate () {
2  var d = new Date();
3  var n = d.getTimezoneOffset();
4  console.log(n)
5}
6myDate();
7
8// PS E:\demo> node 1.js
9// -480

也就是 480 分钟（8 个小时），北京时间是东八区，刚好与格林威治时间相差 8 个小时，这说明 getTimezoneOffset() 方法是很有效的。

接下来利用这个方法，判断一下当前时区是否为夏令时。

 1const time1 = new Date(2021, 0, 1);
 2const time2 = new Date(2021, 6, 1);
 3if (time1.getTimezoneOffset() != time2.getTimezoneOffset()) {
 4  console.log('夏令时');
 5}
 6else {
 7  console.log('非夏令时');
 8}
 9
10
11// PS E:\demo> node 1.js
12// 非夏令时
