将数据库中的图片信息导出并调用

09 Jul, 2021

接口链接：http://121.196.166.173/img/img.php 展示 demo：http://121.196.166.173/img

前言

为了写博客以及其他的一些用途，我使用服务器搭建了一个连接 github 仓库的上传网页，并且在数据库中记录上传的信息，比如缩略名、时间戳和图片链接。分别对应 GitHub 仓库中的图片，但是后期我发现在 GitHub 查看图片非常别扭，因此我打算写一个可展示图片的网页，把 GitHub 仓库中的图片通过链接展示出来，当然我们不可能一张张复制，还好有数据库。

看一下具体内容

写一个接口

很好，拥有我们需要的字段。事不宜迟，直接开动，下面是完成的接口代码。

 1<?php
 2
 3header('Content-Type:application/json; charset=utf-8');
 4header("Access-Control-Allow-Origin:*");
 5
 6$servername = "localhost";
 7$username = "imgbed";
 8$password = "imgbed";
 9$dbname = "imgbed";
10
11// 创建连接
12$conn = new mysqli($servername, $username, $password, $dbname);
13 
14// 检测连接
15if ($conn->connect_error) {
16    die("连接失败：" . $conn->connect_error);
17}
18
19// imgmd5 名称转 md5
20// imguploadtime 上传时间戳
21// imgurl 链接
22// 上传 ip
23$sql = "select imgmd5,imguploadtime,imgurl,imguploadip from remote_imgs
24";
25$result = $conn->query($sql);
26
27
28if ($result->num_rows > 0) {
29    // 输出数据
30    while($row = $result->fetch_assoc()) {
31        
32    $data[]=$row;
33    
34    }
35    
36    $json = json_encode($data,JSON_UNESCAPED_UNICODE|JSON_PRETTY_PRINT);//把数据转换为 JSON 数据。
37    
38    exit($json) ;
39
40} else {
41    echo "未查询到结果！";
42}
43
44$conn->close();
45
46?>

分别将缩略名，时间戳，图片链接，和上传 ip 通过 json 格式导出，非常完美。接口链接：http://121.196.166.173/img/img.php

使用 ajax 进行调用

然后只需要在前端将接口调用，然后简单写一个页面即可，下面是 html 代码，，通过 ajax 调用。

 1<!DOCTYPE html>
 2<html lang="en">
 3
 4<head>
 5    <meta charset="UTF-8">
 6    <meta http-equiv="X-UA-Compatible" content="IE=edge">
 7    <meta name="viewport" content="width=device-width, initial-scale=1.0">
 8    <title>图床</title>
 9    <style>
10        .container {
11            max-width: 1000px;
12            margin: 40px auto;
13            display: flex;
14            flex-wrap: wrap;
15            justify-content: space-between;
16        }
17        .item {
18            width:300px;
19            /* height: 300px; */
20            overflow: hidden;
21            border: 2px solid #bbb;
22            margin-bottom: 24px;
23        }
24
25        .item a{
26            display: block;
27            width: 300px;
28            /* height: 300px; */
29            overflow: hidden;
30        }
31        .item img{
32            max-width: 300px;
33            max-height: 300px;
34        }
35    </style>
36</head>
37
38<body>
39    <div class="container"></div>
40    <script src="https://cdn.zburu.com/list/jquery3.6.0.js"></script>
41    <script>
42        var str = '';
43        $.ajax({
44            url: 'http://121.196.166.173/img/img.php',
45            type: 'get',
46            dataType: 'json',
47            async: false,
48            success: function (data) {
49                $.each(data, function (i, item) {
50                    console.log(item)
51                    list = "<div class='item'><p>缩略名：" + item.imgmd5 + "</p>" +
52                        "<p>时间戳：" + item.imguploadtime + "</p>" +
53                        "<a target='_blank' href='"+ item.imgurl +"'><img src='" + item.imgurl + "'></a>" +
54                        "<p>上传 ip：" + item.imguploadip + "</p></div>"
55
56                    str += list;
57                }),
58                $(".container").html(str);
59                console.log('数据请求成功')
60
61            },
62            error: function () {
63                console.log('数据请求失败')
64            }
65
66        });
67    </script>
68</body>
69
70</html>
