Nuxt 项目中如何引入百度统计？

17 Jan, 2022

在部署一个网站之后，我们最关心的事情无异于是访问量，以及对它的分析。国内的百度统计是一个不错的选择，基本的功能都是免费的。

只需要在 head 中引入一串 javascript 代码即可。

1<!-- <script>
2var _hmt = _hmt || [];
3(function() {
4  var hm = document.createElement("script");
5  hm.src = "https://hm.baidu.com/hm.js?xxxxxxxxxxxxxxxxxxx";
6  var s = document.getElementsByTagName("script")[0]; 
7  s.parentNode.insertBefore(hm, s);
8})();
9</script> -->

上面是百度提供的统计代码，需要放在 <head></head>中。

但是在 nuxt 中，没有传统的 <head></head> 。所以要对他进行一些处理。

1. 首先在根目录下 /plugins 新建一个文件 baidu.js

 1// /plugins/baidu.js
 2
 3export default ({app: {router}, store}) => {
 4  /* 每次路由变更时进行 pv 统计 */
 5  router.afterEach((to, from) => {
 6    /* 告诉增加一个 PV */
 7    try {
 8      window._hmt = window._hmt || []
 9      window._hmt.push(['_trackPageview', to.fullPath])
10    } catch (e) {
11    }
12  })
13}

2. 配置 nuxt.config.js 文件

  1. 在 plugins中：
1plugins: [
2  {
3    src: '~/plugins/baidu'
4  }
5],
  1. 在 head中：
1head: {
2  // ...
3  link: [
4    // ...
5  ],
6  script: [
7    { src: 'https://hm.baidu.com/hm.js?xxxxxxxxxxxxxxxxxxx' }
8  ]
9},

在 script 中写入百度统计提供的 url 即可，按照对应的字符。
