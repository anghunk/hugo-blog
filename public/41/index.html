如何在 React 中引入 less？

04 Jan, 2021

本文主要写如何在 React 中引入 less。因为 less 和 css 非常像，因此很容易学习。而且 less 仅对 css 语言增加了少许方便的扩展，这就是 less 如此易学的原因之一。

1.安装 less

1npm install less less-loader --save-dev

2.暴露 webpack 文件

利用 npx create-react-app 搭建的 React 项目，默认隐藏 webpack 配置文件，引入 less 需要修改 webpack 配置文件，因此我们需要执行命令暴露 webpack 配置文件。

这里需要注意，一旦暴露，无法回退。

1npm run eject

如果这一步失败，执行下列命令：

1git add .
2
3git commit -m "init"

然后再执行 npm run eject

注意：暴露 webpack 文件只能在 create-react-app 之初，一旦项目结构发生改变，再进行暴露操作就会失败。所以尽量在项目建立时就进行 npm run eject 操作。

3.修改 wenpack.config.js 配置

在合适的位置添加：

1// 放在 
2// const sassRegex = /\.(scss|sass)$/;
3// const sassModuleRegex = /\.module\.(scss|sass)$/; 
4// 后面
5
6const lessRegex = /\.less$/;
7const lessModuleRegex = /\.module\.less$/;

 1// 放在 oneof 数组下
 2
 3{
 4  test: lessRegex,
 5  exclude: lessModuleRegex,
 6  use: getStyleLoaders(
 7    {
 8      importLoaders: 2,
 9      sourceMap: isEnvProduction && shouldUseSourceMap,
10    },
11    'less-loader'
12  ),
13  // Don't consider CSS imports dead code even if the
14  // containing package claims to have no side effects.
15  // Remove this when webpack adds a warning or an error for this.
16  // See https://github.com/webpack/webpack/issues/6571
17  sideEffects: true,
18},
19// Adds support for CSS Modules, but using SASS
20// using the extension .module.scss or .module.sass
21{
22  test: lessModuleRegex,
23  use: getStyleLoaders(
24    {
25      importLoaders: 2,
26      sourceMap: isEnvProduction && shouldUseSourceMap,
27      modules: true,
28      getLocalIdent: getCSSModuleLocalIdent,
29    },
30    'less-loader'
31  ),
32},
