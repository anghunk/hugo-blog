在钱塘江北岸遥望。

1.Whois 查询

https://who.cx/

一个美观的 Whois 查询网站，加载速度很快，可以识别备案域名。

2.excalidraw 画板

https://excalidraw.com/

在线画板工具，支持协同、分享等。

3.图形生成器

https://itakeo.github.io/tool/

快速生成三角形、箭头、阴影、渐变的 CSS 代码。对我来说是不可多得的好工具，日常使用会用到 css 三角形，每次手动画总是很麻烦，有了这个工具，以后可以省很多力气了。
