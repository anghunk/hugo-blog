在 js 中运算不能随便把 value 属性定义成变量

18 Nov, 2020

在写一个小 demo 的时候，无意中发现定义变量的时候直接写入 value，会导致获取不到数据，或者获取的不是我们想要的内容，在 js 中运算不能随便把 value 属性定义成变量

先看一段代码

 1<input type="text"" id="a">
 2<span>*</span>
 3<input type="text" id="b">
 4<input type="button" value="=" onclick="beto()">
 5<input type="text" id="sub" disabled>
 6
 7<script>
 8    function beto() {
 9        var a = document.getElementById('a').value
10        var b = document.getElementById('b').value
11        var sub = document.getElementById('sub').value
12        
13        sub = a + b
14    }
15</script>

就这么一看，逻辑貌似没有问题，获取 a 和 b 的 value，乘法运算，然后输出 sub

但是放在浏览器运行的时候，发现完全没有反应

为什么呢？

（假装思考五分钟…）

因为 sub.value 不能直接定义在变量中

1// 先定义 id 为 sub 的 input 框
2var sub = document.getElementById('sub')
3
4// 然后在运算的时候在直接使用 sub.value
5sub.value = a * b

顺手放一个写 demo 时候用的简易计算器

 1<!DOCTYPE html>
 2<html lang="en">
 3<head>
 4    <meta charset="UTF-8">
 5    <title>简易计算器</title>
 6</head>
 7<body>
 8    <input type="text"" id="a">
 9    <select id="c">
10        <option value="+">+</option>
11        <option value="-">-</option>
12        <option value="*">*</option>
13        <option value="/">/</option>
14    </select>
15    <input type="text" id="b">
16    <input type="button" value="=" onclick="beto()">
17    <input type="text" id="sub" disabled>
18   
19    <script>
20        function beto() {
21            var a = document.getElementById('a').value
22            var b = document.getElementById('b').value
23            var c = document.getElementById('c').value
24            var sub = document.getElementById('sub')
25            
26            switch(c) {
27                case  "+":
28                sub.value = parseInt(a) + parseInt(b);
29                break;
30                case  "-":
31                sub.value = parseInt(a) - parseInt(b);
32                break;
33                case  "*":
34                sub.value = parseInt(a) * parseInt(b);
35                break;
36                case  "/":
37                sub.value = parseInt(a) / parseInt(b);
38                break;
39            }
40        }
41    </script>
42</body>
43</html>
