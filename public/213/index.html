hugo 的一些模板语法

15 Nov, 2023

分享一下 hugo 的语法模板，如何获取总文章数和总字数，效果可以参考我的首页。

1{{ $articleCount := len .Site.RegularPages }}
2{{ $totalWordCount := 0 }}
3{{ range .Site.Pages }}
4{{ $totalWordCount = add $totalWordCount .WordCount }}
5{{ end }}

1<p>已经写了 {{ $articleCount }} 篇文章，共 {{ $totalWordCount }} 字。</p>

在首页过滤某个分类。

 1{{ $pages := .Site.RegularPages }}
 2{{ $paginator := .Paginate ($pages) }}
 3{{ $count := 0 }}
 4
 5{{ range $paginator.Pages }}
 6{{ $title := .Title }}
 7{{ $summary := .Summary }}
 8
 9{{ if and (not (in .Params.categories "weekly")) (lt $count 7) }}
10{{ $count = add $count 1 }}
