hugo 纯静态编写一个字数统计脚本

13 May, 2023

早上起来的时候看到微信群中有博友分享自己的建站时间，聊着聊着说到总字数上面，这时我才发现我的网站没有统计的地方，索性就自己写一个吧，因为是静态博客，所以不涉及后端、服务器等，只用前端的方式来解决。

这里需要借助我之前的一篇文章，《用原生 js 给网站写个搜索功能》，里面有提到如何生成一个文章列表 json 文件，这里我们会用到这个文件 index.json。

写一个 ajax 方法，让浏览器遍历每个页面。

 1$(document).ready(function () {
 2  $.ajax({
 3    url: '/index.json',
 4    type: 'get',
 5    dataType: 'json',
 6    success: function (data) {
 7      const pageUrls = data;
 8      const totalNum = pageUrls.length;
 9      $('#totalNum').html(totalNum);
10      let totalWords = 0;
11      pageUrls.forEach(urlObj => {
12        $.get(urlObj.permalink, function(data) {
13          const content = data.replace(/(<([^>]+)>)/gi, " ").replace(/[^\w\s]/gi, " ");
14          const words = content.split(" ");
15          const wordCount = words.filter(word => word !== "").length;
16          totalWords += wordCount;
17          $('#totalWords').html(totalWords);
18        });
19      });
20    },
21    error: function () {
22      console.log('error')
23    }
24  })
25
26});

我是将信息放在我的 关于 页面下。

1经过脚本统计，我已经写了 <span id="totalNum"></span> 篇文章，总共 <span id="totalWords"></span> 个字。
