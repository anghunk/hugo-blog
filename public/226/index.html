vue 中配置 env 文件

24 Apr, 2024

.env 命名规则。

1# 开发环境 .env 文件名
2.env.development
3
4# 开发环境 .env 文件名
5.env.production

在 vue-cli 中，要求环境配置文件必须以 VUE_APP 开头，如下：

1# .env
2VUE_APP_BASE_URL = http://0.0.0.0
3
4# 使用
5const baseURL = process.env.VUE_APP_BASE_URL;

修改 package.json 文件，在运行指令后加上 --mode 标识。

1// package.json
2"scripts": {
3  "dev": "vue-cli-service serve --mode development",
4  "build": "vue-cli-service build --mode production",
5}
