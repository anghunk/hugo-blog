在 vue 中使用 axios 调用数据

08 Jun, 2021

如何在 vue 中完成一次接口的调用？首选就是 axios，方便快捷又好用，支持各种 api ,封装也很方便。

先用 node 安装一下。

1npm install axios

然后在 main.js 文件中引入。

1import axios from 'axios'
2import VueAxios from 'vue-axios'
3
4Vue.use(VueAxios, axios)

以我的一个工程目录作为实例，看一下代码。

 1// ListThere.vue
 2
 3// html 代码
 4...
 5<div>{{ info }}</div>
 6...
 7
 8// js 代码
 9import axios from "axios";
10
11export default {
12  name: "ListThere",
13  data() {
14    return {
15      ...
16    };
17  },
18  components: {
19    ...
20  },
21  methods: {
22    ...
23  },
24  mounted() {
25    axios
26      .get("http://api.h-camel.com/api?mod=interview&ctr=issues&act=today")
27      .then((response) => (this.info = response))
28  },
29};

数据成功的被取到，然后需要将它在页面中展示出来。使用 vue 自带的 v-for 列表渲染。

1<ul class="list-api-show">
2  <li v-for="(item, index) in show" :key="index">
3    {{ item.title }}
4  </li>
5</ul>

然后通过 axios 把数据指向 v-for.

1mounted() {
2  axios
3    // ...
4    .then((response) => (this.show = response.data.result.today))
5}

数据成功取出然后展示在 v-for 里面。

当然了，这是成功的情况，加入失败了呢？我们需要浏览器给出提示，所以 axios 给出了一个 api — err 语法。

 1mounted() {
 2  axios
 3    // ...
 4    .catch(function (error) {
 5      if (error.response) {
 6        console.log(error.response.data);
 7        console.log(error.response.status);
 8        console.log(error.response.headers);
 9        console.log('err')
10      } else if (error.request) {
11        console.log(error.request);
12        console.log('err')
13      } else {
14        console.log('Error', error.message);
15        console.log('err')
16      }
17      console.log(error.config);
18      console.log('err')
19    });
20}

如果接口有问题，或者我们调用时代码写错了，都会给出报错提示，具体什么样的错误会有什么样的警告，需要我们一一去经历摸索。
