Github Actions 自动化打包流程

13 Apr, 2023

Github Actions 监听仓库 commit 事件，然后执行自动化打包流程，并通过第三方平台自动部署，发布非常方便。

 1name: Build
 2on:
 3  push:
 4    branches:
 5      - main
 6
 7jobs:
 8  build:
 9    runs-on: ubuntu-latest
10    steps:
11      - name: Checkout code
12        uses: actions/checkout@v2
13      - name: Install dependencies
14        run: npm install
15      - name: Build
16        run: npm run build
17      - name: Commit changes
18        run: |
19          git config --global user.email "youremail"
20          git config --global user.name "username"
21          git add .
22          git commit -m "Action bot commit"
23      - name: Push changes
24        uses: ad-m/github-push-action@master
25        with:
26          github_token: ${{ secrets.GITHUB_TOKEN }}
27          branch: main

程序会自动获取 github token，只需要在仓库设置 /settings/actions 中把 Workflow permissions 设为读写权限。
