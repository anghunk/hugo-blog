window.scroll 方法只能存在一次该如何解决？

17 Mar, 2021

页面监听是我们经常用的一个功能，下面两个段代码表示两个事件，分别在页面滚动 500px 和 1000px 的时候触发，在代码中使用了箭头函数。

 1window.addEventListener('scroll', () => {
 2   var scrollTop = document.documentElement.scrollTop 
 3   if (scrollTop > 500) {
 4       console.log('滚动了 500px')
 5   }
 6})
 7
 8window.addEventListener('scroll', () => {
 9   var scrollTop = document.documentElement.scrollTop 
10   if (scrollTop > 1000) {
11        console.log('滚动了 1000px')
12   }
13})

但是后面又要求代码兼容到 IE10，就不能采用这种写法了。然后我就想使用原生的function写法。

1window.onscroll = function() {
2    // ...
3}

在用的过程中我发现，如果只有一个window.scroll没有任何问题，但是如果有多个window.scroll，后面的就不会生效。

那么如何解决这个问题呢？

window.scroll只能存在一次，第一个会生效，后面的不生效，可以通过 js 处理这个问题。

 1function addEvent(type, toDo) {
 2    if (window.attachEvent) {
 3        window.attachEvent('on' + type, function() {
 4            toDo.call(window)
 5        })
 6    } else {
 7        window.addEventListener(type, toDo, false)
 8    }
 9}
10
11addEvent('scroll', function(){
12    console.log('第一次调用window.scroll')
13}
14
15addEvent('scroll', function(){
16    console.log('第一次调用window.scroll')
17}

运行一下，多个scroll可以同时运行。
