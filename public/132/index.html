typecho 评论回复艾特评论人

07 Jun, 2022

typecho 在评论时默认是没有 @ 评论人的功能，可以用代码加一下。

在 functions.php 中：

 1/*
 2* 评论回复时 @ 评论人
 3*/
 4function get_comment_at($coid)
 5{
 6    $db   = Typecho_Db::get();
 7    $prow = $db->fetchRow($db->select('parent,status')->from('table.comments')
 8        ->where('coid = ?', $coid));
 9    $mail = "";
10    $parent = @$prow['parent'];
11    if ($parent != "0") {
12        $arow = $db->fetchRow($db->select('author,status,mail')->from('table.comments')
13            ->where('coid = ?', $parent));
14        @$author = @$arow['author'];
15        $mail = @$arow['mail'];
16        if(@$author && $arow['status'] == "approved"){
17            if (@$prow['status'] == "waiting"){
18                echo '<p class="commentReview">（评论审核中）)</p>';
19            }
20            echo '<a href="#comment-' . $parent . '">@' . $author . '</a>';
21        }else{
22            if (@$prow['status'] == "waiting"){
23                echo '<p class="commentReview">（评论审核中）)</p>';
24            }else{
25                echo '';
26            }
27        }
28
29    } else {
30        if (@$prow['status'] == "waiting"){
31            echo '<p class="commentReview">（评论审核中）)</p>';
32        }else{
33            echo '';
34        }
35    }
36}

然后在 comments.php 中输出评论内容代码的前面加上：
